# Python Test Creator
This project was my first large scale programming project, encompassing the full software development lifecycle with iterative stages. It was completed for my A-Level Computer Science coursework.

The project is a Python test creator application, designed for educational institutions. 
The application allows users to register and login as either teachers or students. 
Teachers are able to create tests and give feedback to students. Students are able to complete tests and recieve feedback from teachers.

The application uses `Tkinter` to create several loosely connected windows that contain all application interaction. Additionally, the `os` and `shutil` libraries are used for file management.

The project consists of:
- a single Python file with all code `Test Creator (Final).py`
- multiple prerequisite files required for the code to run: `guide.txt`, `login.csv`, `empty.csv`
- a single 400+ page .pdf report providing analysis, design, development and evaluation documentation `Computing coursework final.pdf` (additionally this document is split into smaller .pdf files)
